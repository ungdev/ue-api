APPLI UTT ARENA

Clique clique clique, c'est l'appli de l'UTT Arena !

Sur cette appli, tu trouveras toutes les infos dont tu as besoin : équipes inscrites, brackets, matchs à suivre, règlements, informations importantes, formulaire de contact... 
Elle est INDISPENSABLE pour comprendre tout ce qu'il se passe pendant le weekend ! Donc activez les notifications !! Les identifiants sont les mêmes qu'à l'inscription ;)

Attention, l'application est optimisée pour les navigateurs PC, sur mobile l'affichage n'est pas adapté L'adresse est simple à retenir : https://uttarena.app !
